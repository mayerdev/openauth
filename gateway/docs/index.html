Submit credentials (login step) or TFA code (TFA step)

Login step: provide method + credentials + password. If TFA is enabled, returns tfa_required: true with tfa_session_id.

TFA step: provide auth_session_id + tfa_session_id + code. Completes authentication and returns the redirect URL.
